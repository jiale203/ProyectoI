Bases de datos Eurostat

Antes de comenzar con el análisis, es muy importante invertir tiempo en la búsqueda de una base de datos fiable y adecuada para obtener las conclusiones deseadas. Por estas razones hemos elegido los datos de Eurostat (https://ec.europa.eu/eurostat), la oficina estadística de la Unión Europea, que proporciona estadísticas y datos de alta calidad sobre Europa.

Gracias a esta web puedes comparar los datos de España con el resto de países europeos, para ver en contexto la situación de la juventud española. Estas son las bases utilizadas donde puedes encontrar todas las variables utilizadas:
